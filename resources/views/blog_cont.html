PURPOSE

The purpose of this exercise is to: Test the functionality of the disassembler, ‘objdump’ (object dump) on your machine from the command line. Use the objdump “-d” option to inspect the machine code object file. Run the linker to generate the executable code.

ACTIVITIES

Please perform each of the following activities. If you have questions, issues, or doubts, please ask for help and do not just guess.

Activity 1: Use the objdump “-d” option to inspect the machine code object file.

Write the following C code in a file name test.c:

int sum(int a, int b, int c) {

int total = a+ b * c;

return total;

}

Generate test.o file, that contains the machine code, by entering the following command on your computer’s command line:

gcc –O1 –c test.c

Disassemble the test.o file, that contains the machine code, by entering the following command on your computer’s command line:

objdump –d test.0

Copy the disassembled output from step c that is in the terminal (console) window

Activity 2: Run the linker to generate the executable code

Write the following C code in a file name start.c, and save in the same directory as test.c:

int main() {

return sum(1, 2, 3);

}

Generate a test.o file, that contains the machine code, by entering the following command on your computer’s command line:

gcc –O1 –c test.c

Generate a linked and executable file called ‘exec’, by entering the following command on your computer’s command line:

gcc –O1 –o exec test.o start.c

The exec file will be much larger after linking; note the size. Disassemble the exec file that contains the machine code, by entering the following command on your computer’s command line:

objdump –d exec

Notice that the file listing also includes code to start and stop the program, as well as other new code. Copy ONLY the ‘sum’ method listing from the disassembled output generated in step d above, from the terminal (console) window:

Compare the output listing in the last step ‘e’ to the output listing in Activity 1 step ‘d’. What is the major difference?

Activities Completed: Save this finished document and upload it to the LMS by 11:59 pm the evening before the next class. Not necessary to upload source code. Make sure you have added your name to the space provided at the top of this document and renamed the document to include your Student ID.
